# GreenCity Tests

## Короткий опис завдання
Цей репозиторій містить тестові кейси для перевірки функціональності сторінки подій та новин на сайті GreenCity.  
Мета тестування — забезпечити коректне відображення подій, роботу фільтрів та можливість створення новин.

## Посилання на сторінку, що тестується
https://www.greencity.cx.ua/#/greenCity/events
